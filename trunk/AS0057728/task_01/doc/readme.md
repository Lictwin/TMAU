# Лабораторная работа №1

## Моделирование управляемого объекта

### Цель работы

Реализовать моделирование изменения температуры управляемого объекта.

---

### Описание

В работе реализованы две модели:

* Линейная модель
* Нелинейная модель

Расчёт выполняется по дискретным моментам времени.
Программа выводит значения температуры на каждом шаге.

---

### Результат

Получены значения температуры для обеих моделей.
Нелинейная модель показывает более сложное поведение системы.

---

### Вывод

Были реализованы и исследованы линейная и нелинейная модели управляемого объекта.
